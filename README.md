# google-ai-content-quality
Google Search's guidance on using generative AI content on your website


Based on Google’s guidance: AI content is acceptable when it helps with research, structure, and useful original content, but mass-generating low-value pages can violate Google’s scaled content abuse policy. Google also says to focus on accuracy, quality, relevance, useful metadata, structured data, alt text, and context about how content was created when appropriate.
